# Auto-Incrementing-Invoice Series
AutoHotkey script designed to automate sequential invoice entry. It eliminates manual typing errors by auto-incrementing complex alphanumeric series (e.g., AAPI0046050S) with a single keystroke, ensuring 100% data accuracy in non-reversible systems
# 🚀 Office Automation & Efficiency Scripts

### The Problem
Manual entry of complex invoice formats (e.g., `AAPI0046050S`) often leads to:
- Typographical errors.
- Irreversible system entries.
- Wasted time on repetitive typing.

### The Solution
A lightweight **AutoHotkey (AHK)** script that automates the sequence.
- **Accuracy:** Eliminates human error by using logic-based increments.
- **Speed:** One-key execution.
- **Flexibility:** Allows manual override of the sequence when needed.

---

## ⌨️ How to Use

1. **Prerequisites:** Install [AutoHotkey](https://www.autohotkey.com/).
2. **Run the Script:** Double-click `InvoiceHelper.ahk`.
3. **Hotkeys:**
   - `F8`: Automatically types the next invoice in the series (e.g., `AAPI0046051S`).
   - `F9`: Opens a prompt to manually update or reset the current sequence number.
